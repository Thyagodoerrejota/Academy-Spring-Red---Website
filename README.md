# Academy-Spring-Red---Website
Projeto de desenvolvimento de um site institucional para academia, com foco em organização estrutural e boas práticas de Engenharia de Software.

## 📌 Sobre o projeto

Este projeto está sendo desenvolvido como prática de desenvolvimento web, aplicando conceitos de estruturação de pastas, separação de responsabilidades e versionamento com Git.

O objetivo é criar um site moderno, responsivo e organizado para uma academia.

## 🚀 Tecnologias utilizadas

-HTML5
-CSS3
-JavaScript (em desenvolvimento)
-Git e GitHub

## 📂 Estrutura atual

-Página Início (estrutura base pronta)
-Página Login (em desenvolvimento)
-Página Planos (em desenvolvimento)
-Página Contato (em desenvolvimento)
-FAG - Formulário de Avaliação Geral (em desenvolvimento)

## 🎨 Tema

O design do site utiliza a cor **orangered** como identidade visual principal.

## 🌍 Idioma

O idioma originário do sistema é o Português-Brasileiro(pt-br).
